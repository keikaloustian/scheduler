## Setup

1. Clone this repository.
2. Install dependencies using the `npm install` command.

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```

## Running Cypress Testing Framework

```sh
npm run cypress
```
